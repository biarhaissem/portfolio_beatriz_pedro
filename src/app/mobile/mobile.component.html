Cantina Reis App

Mobile E-commerce Project

App mobile e-commerce desenvolvido para agilizar o atendimento e diminuir as filas em horários de pico da cantina da universidade.

Utilizamos Figma, React Native, Expo e JavaScript.

01. Reuniões, pesquisas e entrevistas com a cliente e público-alvo (estudantes universitários).

02. Wireframe

03. Fluxo do usuário

04. Protótipos (construídos no Figma)

05. Escolha das cores considerando a cantina física para representação e identificação adequada.

06. Desenvolvimento com React Native e Expo

O aplicativo foi criado e está pronto para as novas etapas de desenvolvimento para integração com e-commerce.

Guerra dos Pets

Mobile Project - Desafio UX/UI

O desafio se baseia na criação do design de um aplicativo, utilizando o Figma, onde serão postadas diariamente fotos de pets, sem edição, para serem votadas por todos os participantes.

Os 5 pets mais votados se consagram campeões do dia.

Etapa 1: Plano Estrutura - Completar Fluxo do Usuário

Etapa 2: Plano Esqueleto - Completar Telas Wireframe

Etapa 3: Plano Superfície - Completar Criação da UI

As atividades acompanharam a evolução dos módulos da Trilha para Elas - Introdução ao Design UX/UI, para as alunas colocarem na prática os aprendizados, como por exemplo, os 5 Planos de Garrett.
